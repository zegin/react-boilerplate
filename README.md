# React-boilerplate

## Stack

### React

### Express

### Jest

### Storybook

### Eslint



# How-to

## Install

```
$ yarn
```

## Start frontend only


```
$ yarn dev
```
*Application running on : [localhost:9000](http://localhost:9000/)*


## Start application
```
$ yarn build
$ yarn start
```
*Application running on : [localhost:3000](http://localhost:3000/)*
