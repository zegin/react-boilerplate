# React boilerplate

This is a opinionated react boilerplate based on my needs

Forked from https://github.com/skyllo/react-hot-express

# Dependencies
- Webpack 4+
- Babel 7+
- React 16+
- React Hot Loader 4+
- Express 4+
- Jest + Enzyme
- ESLint

# Commands
Development
```
yarn dev
```

Production
```
yarn build && yarn start
```

Test
```
yarn test
```

Storybook
```
yarn storybook
```

Lint
```
yarn lint
or
yarn lint:write
```
